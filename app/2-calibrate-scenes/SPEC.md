- 2. カメラ校正ページ（単体）
  - TypeScript の UI と C++ wasm で実装
  - パス：/2-calibrate-scenes
  - 各カメラを「一つずつ」校正します（ペア選択は不要）。
  - UI は対象カメラとモデル（normal/fisheye）を選び、対象カメラのフレームのみを複数選択して実行します。
    - 実行前プレビュー: 対象カメラに対してコントラスト/反転などのプリプロセスを調整し、（任意で）コーナー検出のオーバーレイを表示できます。
    - wasm（OpenCV）で内部・外部パラメータを計算し、1 ファイル（例: `2-calibrate-scenes/<runTs>/cam-<name>_calibration.json`）として保存します。
    - OpenCV の remap により、歪み補正 map を `2-calibrate-scenes/<runTs>/cam-<name>_remapXY.xy` に保存します。
  - カメラ間ホモグラフィの計算は本ページから分離し、/6-cameras-homography で実行します。
